# SwiftUI Components

## UI Patterns
- **Glassmorphism-inspired Cards**: Use `.background(Material.thin)`.
- **Media Grids**: Standard `LazyVGrid` layouts for `CardGrid.swift`.
- **Detail Views**: Consistent header, description, and action buttons.

## Existing Components
- `RatingView`: Modular component for media reviews.
- `GridCard`: Standardized card for all media types.
